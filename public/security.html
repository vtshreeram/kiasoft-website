Security

✓ HIPAA Compliant

At Kiasfot Technologies, operated by Kiasfot Technologies Private Limited, security is not just a feature—it's our foundation. We are committed to protecting your data with the highest standards of security and compliance.

HIPAA Compliance

We fully comply with the Health Insurance Portability and Accountability Act (HIPAA). Our policies, procedures, and technical safeguards ensure the protection of all Protected Health Information (PHI).

End-to-End Encryption

All data is encrypted in transit using TLS 1.3 and at rest using AES-256 encryption. Your sensitive information is never transmitted or stored in plain text.

SOC 2 Type II

Our infrastructure undergoes regular SOC 2 Type II audits to ensure we meet the highest standards for security, availability, and confidentiality.

ISO 27001 Certified

We maintain ISO 27001 certification, demonstrating our commitment to information security management systems (ISMS) best practices.

Our Security Framework

Kiasfot Technologies implements a comprehensive security framework based on industry best practices:

1. Data Protection

  • Encryption: AES-256 encryption for data at rest, TLS 1.3 for data in transit
  • Access Control: Role-based access control (RBAC) with granular permissions
  • Data Segregation: Logical separation of customer data in multi-tenant architecture
  • Backup & Recovery: Automated daily backups with encrypted offsite storage

2. Infrastructure Security

  • Cloud Hosting: Enterprise-grade cloud infrastructure with SOC 2 compliant providers
  • Firewalls: Web Application Firewall (WAF) protection against DDoS and attacks
  • Network Security: VPC isolation and private subnets for sensitive services
  • 24/7 Monitoring: Real-time security monitoring and anomaly detection

3. Application Security

  • Secure Development: Regular security code reviews and penetration testing
  • Vulnerability Scanning: Automated weekly vulnerability scans
  • Input Validation: Comprehensive input sanitization to prevent injection attacks
  • Session Management: Secure session tokens with automatic timeout

4. Compliance & Certifications

We maintain the following certifications and compliance frameworks:

  • HIPAA (Health Insurance Portability and Accountability Act)
  • SOC 2 Type II
  • ISO 27001 (Information Security Management)
  • GDPR (where applicable)
  • India Data Protection Bill compliance

5. Incident Response

We maintain a comprehensive incident response plan that includes:

  • 24/7 security operations center monitoring
  • Rapid incident detection and response procedures
  • Regular security incident drills and tabletop exercises
  • Defined communication protocols for breach notifications

6. Employee Security

  • All employees undergo background checks
  • Mandatory security awareness training
  • Signed confidentiality agreements
  • Principle of least privilege enforced

7. Third-Party Security

We carefully vet all third-party vendors and require:

  • Security questionnaire completion
  • Proof of security certifications
  • Data Processing Agreements (DPAs)
  • Regular security assessments

8. Customer Responsibilities

While we provide robust security, we encourage our customers to:

  • Use strong, unique passwords
  • Enable two-factor authentication
  • Regularly review access logs
  • Report suspicious activities immediately
  • Sign Business Associate Agreements (BAA) for PHI handling
